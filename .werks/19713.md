# omd: Fix calling 'omd status core'

key        | value
---------- | ---

version    | 3.0.0b1
class      | fix
edition    | community

Previously, `omd start core`, `omd stop core`, `omd restart core` and `omd
reload core` worked as expected, but `omd status core` did not:

```
❯ omd status core
ERROR: This daemon does not exist.
```

This has been fixed.
