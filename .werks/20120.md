# Fixed download links on 'Setup > Agents > Other operating systems'

Previously, opening _Setup > Agents > Other operating systems_ offered agent files that are provided by monitoring plugins — for example the Oracle plugin's mk-oracle files.
Clicking one of these files resulted in an error stating that the file could not be downloaded, because the generated download link was malformed.

This has been fixed. These files are now served through a dedicated download handler and can be downloaded from this page again.
