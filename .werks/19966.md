# Windows agent: remove unused pyOpenSSL from bundled Python environment

key        | value
---------- | ---
date       | 2026-07-22T06:40:55+00:00
version    | 3.0.0b1
class      | fix
edition    | cre
component  | checks
level      | 1
compatible | yes

The Python environment bundled with the Checkmk Windows agent no longer includes
the pyOpenSSL package.

pyOpenSSL was an unused dependency and is not imported by the agent or any of its
plugins. Removing it reduces the size as well as the dependency and CVE surface of
the bundled Python environment.

There is no functional change for users.
