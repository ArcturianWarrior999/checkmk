Checkmk now offers guided Quick Setups for ingesting OpenTelemetry and
Prometheus data. You can find them in Setup under the *Telemetry* topic:

- *OpenTelemetry*: guides you through creating an OpenTelemetry collector
  receiver — endpoint IP address and port, encryption, authentication for
  gRPC and HTTP, and forwarding to the Event Console — together with the
  matching dynamic host configuration (DCD) connection, all in a single
  wizard.
- *Prometheus*: guides you through configuring a Prometheus scrape target
  and its linked DCD connection.

Previously these configurations had to be assembled by hand across several
Setup pages. The Quick Setup creates and links the collector or scraper
configuration, the required passwords, and the DCD connection for you. It
shows a summary of everything that will be created before you save, and lets
you edit or delete the whole configuration afterwards from a single overview
page.

The existing OpenTelemetry collector and Prometheus scrape configuration
pages remain available. Configurations created through the Quick Setup are
marked as managed by it, so editing them directly warns you that they are
maintained by the Quick Setup.
