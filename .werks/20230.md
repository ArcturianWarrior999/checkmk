# Host search: Fix label search matching only the label key

key        | value
---------- | ---
date       | 2026-07-15T12:05:58.847607+00:00
version    | 3.0.0b1
class      | fix
edition    | cre
component  | wato
level      | 1
compatible | yes

Previously, searching for hosts by an explicit label under
_Setup > Hosts_ (via the _Search for hosts_ display action) only
compared the label key and ignored the label value. For example,
searching for the label "dc:MM" also returned hosts with "dc:MG",
since both share the key "dc".

This has been fixed. The host search now matches the full label,
so only hosts carrying the exact "key:value" pair are returned.
