# ups_bat_temp: Fix crash when UPS reports an empty battery temperature

key        | value
---------- | ---
date       | 2026-07-08T14:40:32.579403+00:00
version    | 3.0.0b1
class      | fix
edition    | community
component  | checks
level      | 1
compatible | yes

Some UPS devices (e.g. the CS141 SNMP/WEB Adapter) report an empty value for
the battery temperature. This caused the check to crash with
"ValueError: invalid literal for int() with base 10: ''". The check now
skips such devices instead of crashing.
